# NoLimit_Test
Seleksi tes kerja NoLimit Data Engineer

Berikut ini adalah jawaban saya untuk soal yang sudah di berikan sebelumnya oleh NoLimit

1. Untuk soal jawaban nomor 1 itu terdapat pada file
   - foward_proxy.py
   - Untuk menjalankan program tersebut tuliskan "python foward_proxy.py" pada terminal
   - Lalu buka terminal baru dan jalankan command pada soal untuk melakukan testing
   - lalua ada juga file pyhton bernama foward_proxy_bonus.py itu untuk jawaban soal bonus
2. Untuk jawaban soal nomor 2 terdapat pada file
   - scraper.py ini digunakan untuk membuat logika scraping dari wikipedia
   - run_scraper.sh adalah file yang digunakan untuk menjalankan program scraper.py
   - untuk menjalankan program pertama kita harus menuliskan ini pada terminal "chmod +x run_scraper.sh"
   - lalu selanjutnya tuliskan command  ./run_scraper.sh "Lionel Messi"  seperti berikut ini untuk menjalakan scraping dari wikipedia
   - Note nama pharse "Lionel Messi" itu bisa di ubah sesuai dengan data keyword yang ingin kita scraping
